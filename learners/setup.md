## What You Need for Session A

Sessions 1a, 2a, and 3a form the **QGIS and Raster Data** track. Complete the items below before the first session begins. If you already finished the main workshop setup page, you are ready.

### Software

**QGIS Desktop (LTR)** — download and install from [https://qgis.org/download/](https://qgis.org/download/). Open QGIS once to confirm it launches. You can check out a quick guide to QGIS [here](https://spatialturn.github.io/intromoduletest/qgisintro.html).

### Accounts

- **USGS EarthExplorer** — for downloading CORONA imagery and SRTM elevation data (Session 2a) → [Register here](https://ers.cr.usgs.gov/register)
- **Copernicus Data Space** — for downloading Sentinel-2 satellite imagery (Session 2a) → [Register here](https://dataspace.copernicus.eu)

Both accounts are free. Email verification can take a few minutes, so please register ahead of time.

### Project Folder

Create a folder on your desktop called **Session_1a**. You will save all downloaded data and QGIS project files here. Keeping everything in one place prevents broken layer links.
